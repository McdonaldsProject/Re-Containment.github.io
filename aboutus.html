ABOUT US

We First Got The Idea To Start Our Server In August Of 2020. When One Of Our Mods Tried Hosting A Server Locally, We Had The Idea Of Making A Dedicated Server. After Some Hard Work, We Had Created A Fully Functional Server, So We Decided To Name It: Re-Containment. Our main goal with Re-Containment is to change the way that SCP Secret Laboratory is played, using unique plugins and configurations.
